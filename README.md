# JSON to CSV Converter (json2csv)

## Descrição

O **json2csv** é uma biblioteca JavaScript utilizada para converter dados no formato **JSON (JavaScript Object Notation)** para **CSV (Comma-Separated Values)**.  
Ela é muito usada em APIs, sistemas web e relatórios para exportar dados em formato compatível com **Excel** e **Google Sheets**.

---

## O que é JSON?

JSON é um formato leve e estruturado para troca de dados entre sistemas.

---

## O que é CSV?

CSV é um formato de texto onde os dados são separados por vírgulas ou ponto e vírgula, muito usado em planilhas.

---

## Vantagens do json2csv

- Conversão rápida de JSON para CSV

- Fácil integração com Node.js e front-end

- Compatível com Excel e Google Sheets

- Permite customizar colunas e delimitadores

- Ideal para exportação de dados e relatórios

---

## Aplicações Práticas

- Exportação de dados de APIs

- Sistemas de estoque

- Dashboards e relatórios

- Ferramentas de conversão de arquivos

- Backup de dados em CSV
